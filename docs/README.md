# 구현할 기능 목록

1. 입력
   - 서로 다른 3자리의 숫자 입력
     - 예외 상황
       - 같은 숫자를 입력할 경우
       - 3자리 숫자가 아닌 경우
       - 숫자가 아닌 문자를 입력할 경우
   - 종료 혹은 재시작 입력
     - 예외 상황
       - 1 혹은 2가 아닌 다른 숫자를 입력할 경우
       - 숫자가 아닌 문자를 입력할 경우
   - 잘못된 값을 입력할 경우 `IllegalArgumentException` 발생
     - 해당 branch에서는 그 지점에서 다시 입력받는다.

<br>

2. 숫자 야구 게임
   - 서로 다른 3개의 랜덤한 숫자 선택
   - 사용자의 입력에 따른 게임 결과 출력
     - 같은 수가 자리에 있으면 `스트라이크`
     - 다른 자리에 있으면 `볼`
     - 같은 수가 전혀 없으면 `낫싱`

3. 출력
   - 게임 결과 출력
   - 재시작 혹은 종료 메시지 출력
